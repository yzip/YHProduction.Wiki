### 工作原理

GoElites 使用了 *合集* 的理念, 这表示所玩家所需要的一切全部包含在里面了，无需在加载额外的设置. 每一个模块都是高度可定制化的，根据用户的需求可以在游戏菜单内进行修改。当然每一个独立的模块也能完全被关闭，这让 GoElites 减少对于 CPU 以及内存的使用.

### 默认热键


| 按键 | 动作 |
|--|--|
| **左 Shift** | 开启 GoElites 游戏内菜单. |
| **空格** | 连招. |
| **X** | 补兵. |
| **C** | 骚扰. |
| **V** | 清线. |
| **Z** | 逃跑. |
| **NUM 1** | 连招不移动. |
| **F9** | 重新加载 GE. |
| **鼠标左键** | *点击与敌方英雄*: 集火的目标; *点击任意位置*: 取消集火. |

### Visual（线圈）

| 设置 | 用途 |
|--|--|
| **Disable all drawings** |屏蔽一切线圈. |
| **Tower ranges** | 防御塔附近显示塔攻击范围.|
| **Zoom Hack** | 无限视距 (*游戏默认滚轮进行调整*). |
| **Glow Enabled** | 启用发光效果. 需要开启此项目其他发光功能才会显示. |
| **Glow My Hero** | 在战场上自身发光更好的看到自己在哪. |
| **Glow Minions** | 高光可被补兵的小兵. |
| **Glow Selected Target** | 高光集火选择的目标. |

### Skin Changer（换肤）

如果启用，将会替换当前的皮肤到菜单中所选的皮肤. 
**注意**: 这只能自己看见而已，本地端而已.

### Evade （躲避）

| 设置 | 用途 |
|--|--|
| **Config**: Collision Mode | 检查体积碰撞. 可以用来检查小兵或者英雄. |
| **Config**: Block Spells Mode | 使用技能来躲避而非移动. |
| **Config**: Block Attacks Mode | 使用技能来阻挡普通攻击而非移动 (比如: *潘僧被动*).  |
| **Config**: Force Fastest Point | 选择最快捷径来躲避技能,无视其他情况. |
| **Config**: Reaction time | 躲避前延迟（反应时间）. |
| **Config**: IgnoreReactionTime | 无视反应时间的开关. |
| **Config**: Disable circular spells dodging | 只躲避圈形技能开关. |
| **Config**: Disable FOW dodging | 无视躲避来自战争迷雾的技能开关. |
| **Config**: Disable dodging if shielded | 有法术盾时不躲避开关. |
| **Config**: Double click to remove | 开启双击在技能线圈就不躲避的功能. |
| **Config**: Enable Evade | 躲避开关 ON/OFF. |
| **Config**: Dodge Only Dangerous | 只躲避危险的技能开关. |
| **Evading spells**: SpellName | 如果不可移动躲避时启用 SpellName 来躲避. 危险等级要设定好才能使用 SpellName 来躲避. |
| **Spells**: SpellName | 只躲避一些技能开关. 危险等级可以在这里被调整. 就算技能被屏蔽了,还是一样会显示线圈除非关闭显示技能. |
| **Humanizer settings**: Spell Detection Delay |设置检测技能的延迟. |
| **Humanizer settings**: Maximum angle | 设置最大角度在于你的移动路径和躲避路径之间. |
| **Humanizer settings**: Dont Dodge If Alt-Tab | 切出游戏时屏蔽躲避. |
| **Humanizer settings**: Ignore Skillshots Not Visible OnScreen | 技能不在屏幕上可视时不躲避开关. |
| **Humanizer settings**: Dont Dodge If Player Is Not Visible OnScreen | 你的英雄不在屏幕上可视时不躲避. |
| **Humanizer settings**: Ignore FOW Exit | TBA | 
| **Humanizer settings**: Minimal Visiblity Time | TBA |
| **Drawing settings**: Line Width | 躲避线粗. |
| **Drawing settings**: Draw Evade Status | 显示躲避开关. |
| **Drawing settings**: Draw Evade Point | 显示躲避方向. |
